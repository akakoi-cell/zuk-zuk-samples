# Claude Design 依頼文 — `Hello Tree` 子供英会話教室サンプル（アーカイブ）

> https://claude.ai/design を**別タブ**で開き、 下の「📋 貼り付け用プロンプト」 をそのまま貼って実行する。
> 出力された HTML/CSS は `claude-design-school/`（salon と同様）に保存し、 CLI 自走（night-run-004）の実装見本にする。
> 設計の根拠: `sample-school-design-brief.md` / `sample-school-hearing.md`

---

## 使い方

1. claude.ai/design を別タブで開く
2. 下の「📋 貼り付け用プロンプト」 を全文コピペして送信
3. 生成されたデザインを確認 → 気に入ったら Hero コピーや色味を微調整
4. エクスポート（HTML/CSS）→ `claude-design-school/` に保存
5. `night-run-004.md` の実装でこのデザインを「見本」 として参照

## 📋 貼り付け用プロンプト

```
日本の「子供英会話教室」のランディングページを1ページ、デザインしてください。

# ブランド
- 教室名: Hello Tree（ハローツリー）English School
- 場所: 埼玉県さいたま市・北浦和の住宅街にある、個人経営の小さな教室
- 対象: 1.5歳〜小学6年生のこども（決裁者は30-40代の保護者）
- コンセプト: 「ネイティブの会話を、まちがえてOK の空間で。」
  失敗を恐れず、たくさん話して、英会話を楽しめる場所
- 代表: トロント育ちの日加ハーフ女性（30代・行動力があり、教材も自作）

# デザインの方向性（最重要）
「ポップ＆ウォーム」= 子供がワクワク × 親が安心。
明るい・あたたかい・角丸・イラスト主体。木モチーフ（葉・幹・小鳥）を効かせる。
ただし多色を使っても安っぽくならないよう、ベースはクリーム白で上品に保つこと。
手書き風のアクセント（"Hello!" など）や吹き出し、点線、star、ブロブ（有機的な形）を散らす。
チェーン塾のようなケバさは避け、個人教室の手づくり感・あたたかみを出す。

# トンマナ参考（重要）
キッズ珠算会 https://kids-shuzankai.com/ のトーンを参考に。
キーワードは「親しみやすさ × 信頼感」。丸ゴシック、角丸ボタン、教室モチーフのイラストを多用。
比重は保護者向け7 : 子供向け3 を意識し、かわいさだけに振り切らず、
少人数・講師の質・実績・保護者の声など「安心材料」をしっかり見せること。

# 参考画像（添付）
別途添付する hero 写真（明るい緑の壁の教室で、女の子が先生とハイタッチして笑っている）の
色温度・あたたかさ・教室の世界観に調和させてください。実際のサイトもこの写真を Hero に使います。
クリーム白とリーフグリーンを基調に、この写真がなじむ配色・余白にすること。

# カラーパレット
- ベース背景: クリーム白 #FFFDF7
- カード背景: 黄ベージュ #FFF6E4
- メイン: リーフグリーン #4FA85A
- アクセント: サンイエロー #FFC83D / コーラル #FF6B5B / スカイブルー #4FB6E6
- テキスト: こげ茶 #3D3128
※ 差し色（黄/赤/青）は面積を抑え、アクセントのみに使う。グリーンとクリームが主役。

# フォント
- 日本語: 丸ゴシック（Zen Maru Gothic のような、あたたかく丸い）
- 欧文見出し: Baloo 2 / Fredoka のような丸くポップな書体
- 欧文本文: Quicksand / Nunito
- 手書きアクセント: Caveat（"Hello!" などのひとことに）
- 角丸は大きめ（カード24px、ボタンはピル型）

# セクション構成（この順番で）
1. Hero — メインコピー「まちがえても、だいじょうぶ。/ トロント育ちの先生といっしょに、/ たくさん失敗して、たくさん話す。」+ Hello Tree ロゴ + 子供がレッスンを楽しむ写真 + 「無料体験レッスン」ボタン
2. About「まちがえることが、いちばんの近道。」+ 選ばれる理由4つ（少人数4〜6名 / ネイティブ級の代表 / まちがえてOKの空間 / おうちで続く会員サイト）をアイコン付きカードで
3. Courses — 年齢別4コース（親子1.5-3歳 / 幼児 / 小学生低学年 / 小学生高学年）を色分けカードで
4. Teachers — 講師3名（代表Emily を主役に大きく、ネイティブMike、バイリンガルAya）
5. Lesson — レッスンの1日を4ステップのタイムラインで（あいさつ→歌&ゲーム→スピーキング→ふりかえり）
6. Members — 会員マイページ＆おうち学習コンテンツの紹介（2枚のプレビューカード：保護者マイページ / 動画・PDF教材）
7. Voice — 保護者の声4件を2x2カードで（star付き）
8. Pricing — 月謝プラン（週1 ¥8,800 / 週2 ¥15,400、税込）を2カードで、週2に「人気」リボン
9. Trial — 無料体験レッスン予約（カレンダー風UI）
10. Access — 教室情報 + 地図エリア
11. FAQ — アコーディオン
12. Footer — LINE友だち追加バナー（緑）+ コピーライト

# トーン参考
やさしく、たのしく、信頼できる。子供の習いごとサイトだが、保護者がちゃんと「ここに通わせたい」と思える品もあること。

レスポンシブ（スマホ・PC）対応で、まずデスクトップ版をしっかり作ってください。
```

## 微調整のヒント（生成後）

- Hero コピーが弱ければ、 ヒアリングの「案 2（保護者目線）」 も試す
- グリーンが強すぎたら、 クリーム/ベージュの面積を増やすよう指示
- コースの 4 色（黄/コーラル/グリーン/青）が散らかって見えたら、 「彩度を 1 段下げて」 と指示
- 木イラストは「主張しすぎない、 余白の装飾として」 を強調
