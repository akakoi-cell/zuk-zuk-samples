# 美容室サンプル ヒアリング結果 — `& moi` (アンドモイ)

> 参考サイト: https://lamune-kyoto.com/

## 基本情報

| 項目 | 値 |
|---|---|
| **店名** | **`& moi`** (アンドモイ) |
| **店名の由来** | 「と私」 (フランス語の moi = 私) / 「私のための時間」 「私と髪との対話」 のニュアンス |
| **店舗タイプ** | 隠れ家的、 住宅街型、 路面店、 完全予約制 |
| **立地** | 神奈川県川崎市中原区、 **武蔵小杉駅徒歩 7 分の住宅街** (架空住所) |
| **オープン年** | 2018 年 (今年で 8 周年) |
| **店主** | 山田 真理子 (40代女性、 美容師歴 18 年、 元都内大型サロン副店長) |
| **スタッフ** | 3 名 (店主 + 中堅 2 名 / 30 代前半 + 20 代後半) |
| **席数** | 3 席 (各席間に仕切り、 半個室感) |
| **営業時間** | 10:00-19:00 (最終受付 17:30) |
| **定休日** | 月曜日、 第 3 火曜日 |
| **価格帯** | カット ¥6,000〜 / カラー ¥8,000-11,000 / トリートメント ¥3,000- / ヘッドスパ ¥4,000- |

## コンセプト

> **「家でもない、 職場でもない、 もう一つの居場所」**

- 30-40代女性 (働く女性 + 子育て世代) がターゲット
- 「庶民的に通いやすい価格帯」 + 「インテリアと時間にこだわる」 ギャップ
- タワマン住民・武蔵小杉エリアの感度高い女性が「自分のための時間」 を取りに来るサロン

## 訴求ポイント (5 つ)

1. **完全予約制 — 1 名様だけのゆったり時間**: 急かさず 2 時間貸切
2. **オーガニックカラー扱い**: 髪と地球に優しい、 ノンジアミン選択可
3. **ヴィンテージ家具とドライフラワー**: BGM はジャズ、 雑誌が読める空間
4. **4D カット技術**: 骨格・髪質・ライフスタイル・トレンドを読む
5. **1 階路面店、 ガラス張り**: 明るく入りやすい、 武蔵小杉駅徒歩 7 分

## デザイン方向性

### 参考サイト (主)
- **https://lamune-kyoto.com/** ← 京都のラムネ美容室
  - 「モダン × 親密 × 高級感控えめ」 が完全に一致
  - フォント: sans 中心
  - カラー: 明度高い淡色 + 余白活用
  - セクション番号 (01 / 02 / 03)
  - About は 4 枚正方形グリッド (インテリア + 小物)
  - Stylists は 3 名横並び (顔写真 + Stylist + 英名 + 日本名)
  - Menu は「¥4,100〜」 形式の幅表現
  - Gallery は正方形サムネイル密集

## セクション構成 (9 セクション)

```
01. Hero        — テキスト主役 (Hero コピー 案 2) + ロゴ大 + 背景画像 1 枚 (店舗外観 or 入口)
02. About       — インテリア 4 枚 正方形グリッド + コピー (店主写真なし、 インテリア強調)
03. Stylists    — スタッフ 3 名 横並び (顔写真 + Stylist + 英名 + 日本名 + プロフィール短文)
04. Menu        — カット ¥6,000〜 形式、 セクション分け (CUT / COLOR / TREATMENT / SPA)
05. Gallery     — 施術後写真のみ、 正方形サムネイル密集 (12-20 枚 / Sanity 駆動)
06. Voice       — お客様の声 3-5 件
07. Access      — 住所 + 営業時間 + 電話 + 簡易地図 (画像 or iframe)
08. FAQ         — 予約 / キャンセル / 駐車場 / キッズ 等
09. Contact     — 予約フォーム disabled + AI STUDIO 誘導 CTA
```

## Hero コピー (確定: 案 2 空間ベース)

```
住宅街の角、 ヴィンテージの椅子が並ぶ小さなサロン。
窓辺のドライフラワー、 古い雑誌、 ゆるやかに流れるジャズ。
ここに来る日は、 髪のためというより、 私のための時間。
「& moi」 ── 「と私」 の、 ささやかな約束です。
```
